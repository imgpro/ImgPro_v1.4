+ Возможность заменять/добавлять несколько файлов за раз.
+ Возможность извлекать файлы

  увеличении количества файлов в архиве) повреждался первый (по смещению) 
  файл в архиве, и игра могла вылетать.

+ Добавлен флажок OnTop - окно программы поверх всех окон

+ Добавлена кнопка Rebuild - выполняет пересборку архива. 
  Помогает при вылетах из игры и при перепутанных текстурах.

В нижней строчке надо указать путь к *.img файлу. Для этого надо нажать 
кнопку  Browse...  и указать путь к файлу. Также можно просто перетащить 
соответствующий *.img файл в окно программы


Кнопка Add... добавляет файлы в список. Также можно просто перетащить 
нужные файлы в окно программы и они добавятся в список


Extract... - извлекает в указанную папку файлы с теми именами, которые 
есть в списке. Если в окно программы перетащить папку, то при следующем 
извлечении будет предложено извлечь файлы в эту папку


Replace - добавляет в IMG архив файлы из списка, если файл уже есть в 
архиве, он заменяется


=============================================================================


Эта прога отличается от других подобных тем, что не отображает
содержимое архива.

Если вы хотите заменить модель машины, то сначала нужно указать
путь к файлу gta3.img (он лежит в папке models главной папки игры) - 
для этого нажимаем кнопку Browse

Затем в окно программы перетаскиваем или добавляем кнопкой Add 
нужные dff и txd файлы, (они появятся в списке заменяемых файлов)
и нажимаем кнопку Replace

Чтобы в GTA3 было удобнее заменять машины, текстуры и т.п.,
надо сделать так, чтобы в папке GTA3\models файлы 
с именами txd.img и txd.dir
были пустыми (т.е. были размером 0 байт).
После этого надо будет заменять
файлы только в gta3.img

История версий

1.4
 - Добавлена кнопка Rebuild - выполняет пересборку архива
   (иногда помогает при вылетах из игры и, возможно, при перепутанных текстурах)
 - Добавлен флажок OnTop - окно программы поверх всех окон
 - Если перетащить в прогу файл с расширением *.img, он станет текущим архивом
 - Можно перетащить папку в окно программы, и она станет текущей папкой для 
   извлечения файлов
 - Можно изменять размеры окна
 - Появилась кнопка * с помощью неё можно зайти в настройки
 - В настройках можно поменять параметры работы с архивом и сделать их 
   такими же, как у IMG Tool 2.0 или как у Crazy IMG Editor 1.0
 - В настройках можно выбрать сохранять или не сохранять положение и размеры окна
 - Можно ассоциировать IMG-файлы с программой

1.3
 - Исправлено: в версии 1.2 при добавлении нескольких новых файлов (при 
   увеличении количества файлов в архиве) повреждался первый (по смещению) файл 
   в архиве, и игра могла вылетать.

1.2
 - Исправлено: прграмма не работала без некоторых библиотек DLL.


* Если в GTA:SA с помощью IMG Pro 1.2 добавить в gta3.img больше трёх новых файлов 
(тех которых раньше не было в архиве, то есть количество файлов в архиве будет увеличено), 
то первый (по смещению) файл bbb_lr_slv1.dff будет повреждён и, возможно, игра иногда 
будет вылетать. В версии IMG Pro 1.4 этот косяк исправлен.
